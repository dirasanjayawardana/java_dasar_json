# JSON (Java Script Object Notation)
Adalah salah satu format pertukaran data yang mirip seperti Object di JavaScript.
Tipe data JSON :
- Object
- Array
- Number
- String
- Boolean
Untuk tipe data lain seperti Date, bisa menggunakan String.

# JSON Library in Java
- Jackson
- GSON
- FastJSON
- dll
